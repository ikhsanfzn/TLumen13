# Project: Lumen Client App

# Tugas 13

## Project setup

```
composer install
```

### Database: Migrations

```
php artisan migrate
```

### Database: Seeding

```
php artisan db:seed
```

### Local Development Server

```
php -S localhost:5000 -t public
```

## End-point: Get Posts

### Method: POST

> ```
> localhost:5000/public/posts
> ```
